# Sistema de Transporte Urbano - FUP UFC

O sistema de transporte urbano é um projeto desenvolvido para a disciplina de Fundamentos de Programação da Universidade Federal do Ceará (UFC). O objetivo do projeto é criar um sistema que gerencie as rotas, horários e tarifas dos transportes urbanos na cidade. Tudo que envolve esse sistema é feito com base em dados e informações fictícias, ou seja, não tem relação com a realidade.

O projeto foi atribuído pelo professor, mas pensado e planejado por mim. A análise de requisitos foi feita com auxílio do Claude.ai e pode ser conferida em [docs/analise-de-requisitos.md](docs/analise-de-requisitos.md)

Para apresentação do projeto, utilizei um slide gerado também com auxílio do Claude.ai, que pode ser conferido em [docs/slide-de-apoio.pptx](docs/slide-de-apoio.pptx).
